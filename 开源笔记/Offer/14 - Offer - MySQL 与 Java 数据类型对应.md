VARCHAR = String

INT = Integer

DATETIME = Date

DATE = Date

TEXT = String 

DECIMAL = BigDecimal



(x) 不管英文或中文都是存储 x 个

VARCHAR(32) 背景颜色

VARCHAR(64) id

VARCHAR(128) 评价内容

VARCHAR(255) 返回地址

VARCHAR(1024) 头像地址



11 是显示宽度 当插上数据没超过11位  左边用零填充 超过正常显示

INT(11) 金钱 

TEXT 内容



DATE 生日

DECIMAL 打折



DECIMAL(4,2) 代表 4位数 2位小数 比如 99.99 ~ 00.00
